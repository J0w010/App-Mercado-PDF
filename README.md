# App-Mercado-PDF

Aplicativo Windows para geração de etiquetas de preço para mercado em PDF.

## Funcionalidades
- Cadastro de múltiplos produtos
- Geração de etiquetas nos tamanhos A4, A5 e A6
- A6 em modo paisagem
- Um PDF por tamanho com cada produto em uma página separada

## Tecnologias
- Java 17
- JavaFX 21
- iText 7
- Maven

## Como executar
1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Rode `mvn package`
4. Execute o projeto
